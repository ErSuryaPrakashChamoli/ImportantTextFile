*******************************************Local Selection
• In Replace mode, the existing selection is replaced
by the new one.
• In Add mode (also set if you press SHIFT ), the
new selection augments the existing one.
• In Subtract mode (CTRL ), the new selection
is subtracted from the existing one.
• In Intersect mode ( SHIFT+CTRL ), the final
selection contains only the areas common to
both the existing and the new selections.
