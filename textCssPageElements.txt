2. Techniques for Centering Elements on a Web Page
To center text in a block-level element, use the text-align property:
h1, h2, h3 {
text-align: center;
}


3.h1, h2, h3 {
margin-left: auto;
margin-right: auto;
width: 300px;
}

 margin-left and margin-right properties to auto (along with a value
for the width property), you center the element inside its parent element.
